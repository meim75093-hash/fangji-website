# Fen Fang Ji (粉防杞) Official Website

A modern, responsive website for Fen Fang Ji brand, showcasing traditional Chinese herbal products.

## 🌿 About Fen Fang Ji

Fen Fang Ji (粉防杞) is a premium brand specializing in traditional Chinese herbal supplements, focusing on health and wellness products derived from time-tested natural ingredients.

## 📦 Deployment

This website is designed to be deployed on **GitHub Pages**.

### Quick Deploy

1. Create a new repository on GitHub
2. Push all files to the repository
3. Go to Settings → Pages
4. Select `main` branch as source
5. Your site will be live at: `https://YOUR_USERNAME.github.io/fangji-website/`

### Manual Deployment

```bash
# Initialize git repository
git init

# Add all files
git add .

# Create initial commit
git commit -m "Initial commit - Fen Fang Ji official website"

# Add remote repository (replace with your repo URL)
git remote add origin https://github.com/YOUR_USERNAME/fangji-website.git

# Push to GitHub
git push -u origin main
```
